# Portfolio

## Run it

```bash
npm install
npm run dev
```

Then open the local URL VS Code / the terminal prints (usually http://localhost:5173).

## Edit your details

Open `src/Portfolio.jsx` and edit the `CONFIG` object near the top of the file
(name, bio, email, skills, projects, etc).

## Build for deployment

```bash
npm run build
```

This outputs a static `dist/` folder you can deploy anywhere (Vercel, Netlify, GitHub Pages, etc).
